# 🎓 Smart StudyMate AI

An AI-based student assistant that helps in studying efficiently by summarizing notes, extracting key topics, generating quiz questions, and converting summaries into speech.

## 🚀 Features
- Summarizes large notes or PDF files
- Extracts important keywords and topics
- Generates short quiz questions for revision
- Converts summaries into speech
- Clean, interactive web UI built with Gradio

## 🧩 Tech Stack
- Python
- Transformers (BART Model)
- Gradio for UI
- NLTK for NLP tasks
- gTTS for Text-to-Speech
